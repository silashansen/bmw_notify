##BMW LOCK NOTIFY##

###To run this###
Create a new file called bmw_notify.env and fill it with the following:
<pre>
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
BMW_USERNAME=
BMW_PASSWORD=
SMS_FROM=<your twilio 'from' number>
SMS_RECEIVERS=<comma separated list of receivers>
SEND_SUCCESS_NOTIFICATION=<debug flag>
</pre>
